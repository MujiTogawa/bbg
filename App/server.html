参考：Pages 服务提供者名单

  • GitHub Pages

    • 官网：https://pages.github.com/

    GitHub Pages is a static site hosting service that takes HTML, CSS, and JavaScript files straight from a repository on GitHub, optionally runs the files through a build process, and publishes a website.

  • Gitee Pages

    • 官网：https://gitee.com/help/articles/4136

    Gitee Pages 是一个免费的静态网页托管服务，您可以使用 Gitee Pages 托管博客、项目官网等静态网页。

  • CODING 网站托管服务

    • 官网：https://help.coding.net/docs/pages/intro.html

    CODING 网站托管服务是 CODING 联合腾讯云 Serverless 团队，为开发者提供的便捷、稳定、高拓展性的网站托管服务，提供完整、高效的部署流程。无需自建服务器，即可一键部署网站应用，同时支持动态网站和静态网站托管。

  • Cloudflare Pages

    • 官网：https://developers.cloudflare.com/pages/

    Cloudflare Pages is a JAMstack platform for frontend developers to collaborate and deploy websites.

  • Vercel

    • 官网：https://vercel.com/

    Vercel is the best place to deploy any frontend app.

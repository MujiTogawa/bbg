# BBG

## 功能简介

BBG 是一个基于 Electron 开发的静态博客生成器。

## 程序包下载地址

本软件支持 macOS、Windows、Linux 平台。

点击[这里](/zh-cn/download.md)以查看可用的程序包下载地址。

- [更新日志](/zh-cn/CHANGELOG.md)

## 如何使用

下载程序包后安装，然后运行 `bbg` 即可开始使用。

# BBG

[安全提示：请更新到 BBG 20220213 版本](/zh-cn/Security_20220213.md)

## 功能简介

BBG 是一个基于 Electron 开发的静态博客生成器。

## 程序包下载地址

本软件支持 macOS、Windows、Linux 平台。

点击[这里](/zh-cn/download.md)以查看可用的程序包下载地址。

## 如何使用

下载程序包后解压，然后双击 `bbg` 可执行文件即可开始使用。
